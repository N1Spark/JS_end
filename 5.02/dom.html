Document Object Model(DOM)

DOM - концепция згидно з якою вси HTML елементи видображаються на объекти (ООП) JS. Це видображення двобичне: змина HTML видбуваеться на структури объектив, а змина струтури объектив - на HTML. Иншими словами, немае ризници як манипулювати сторинкою - через вплив на HTML чи шляхом змини объектив(и колекции).
