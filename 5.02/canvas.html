Полотно. Засоби рисування

Основу для створення складних зображень дає елемент < canvas> - полотно. З урахуванням особливості DOM - перезбирати весь документ після довільної зміни - полотна надають покращені можливості створення динамічного контенту.
Лінії. Ідея рисування ліній полягає у комбінації двох операцій - переміщення "каретки" без сліду, та переміщення зі слідом.
За замовчанням розмір полотна 300 X 150 пікселів. При цьому важливо розуміти, що є розміри, а є роздільна здатність. Якщо стилями змінити розмір полотна, то його роздільна здатність не зміниться, при збільшенні проявиться піксельна структура.

JavaScript у браузері є ряд обмежень, зокрема, доступ до файлової системи. Для задач збереження даних браузер надає декілька засобів, доступних для вкладці "Application" інструментів розробника (консолі):

  • Local Storage - постійне сховище
  • Session Storage - тимчасове сховище
  • Indexed DB - для великих даних типу "ключ-значення"
